## Spec Summary

Configures Retrofit + OkHttp to communicate with the ExerciseDB API on RapidAPI.
An `AuthInterceptor` injects the `X-RapidAPI-Key` and `X-RapidAPI-Host` headers on
every request. The Koin `networkModule` provides a singleton `OkHttpClient` (with
10s connect / 30s read / 15s write timeouts) and a singleton `Retrofit` instance
backed by a `kotlinx.serialization` JSON converter. `HttpLoggingInterceptor` is
conditionally added only when `BuildConfig.DEBUG` is true, preventing a compile
error in release builds where the dependency is `debugImplementation` only.
